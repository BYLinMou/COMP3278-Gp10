# Features

## Social Features

- User registration and login
- Session recovery through signed cookies
- Profile pages with bio and activity statistics
- Follow and unfollow creators
- Image post creation with upload or image URL input
- Feed browsing with recent and popular sorting
- Category-based filtering
- Like and comment interactions
- Browsing history
- Notification inbox for follows, likes, and comments

## Query and Discovery Features

- Read-only SQL execution
- Deterministic text-to-SQL mapping for supported prompts
- Search comparison between SQL approaches
- AI-assisted discovery agent with explicit approval before execution

## Analytics Features

- Overview metrics
- Top posts
- Active users

## UI and Experience Notes

- Art Deco-inspired design system
- Responsive layout
- Masonry-style content presentation
- Drawer-based post detail interaction
